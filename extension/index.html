This act as the client side to use the project in various e-commerce website. For the time we have extensions for Mozilla Firefox and Google Chrome

Download Project

Download the Opinator Extension project from Github.

You can also use git to clone the repo to your local system using:

$ git clone https://github.com/SaptakS/opinator-plugin.git

Chrome Extension

  • Open google chrome browser
  • Go to url chrome://extensions
  • Check the Developer Mode checkbox
  • Click on Load unpacked extension.. and upload the Chrome Extension folder from the downloaded/cloned folder
